inirt: Quick Start Guide

Explanatory Item Response Theory Tools in R

The inirt project exists to create comprehensive exploratory item response theory (IRT) modeling tools in R.

This website contains the complete package documentation. We include extensive tutorials that cover every topic as well as pages for each user available function.

Installation/Setup

1. Since this is an R pacakge, the user first needs to install R from the R project website.

2. Once R is installed, open an R console and install the devtools package with: 

# Install devtools from CRAN
install.packages("devtools")

3. Then, library devtools and use it to install R package inirt:

# Install inirt from Github
install_github("inirt/inirt")

4. Check that the new package is installed by librarying it in R: 

# Load the new package in R
library(inirt)

If you see no error message, then the package is installed.

5. If the package fails to install or there is a bug please open an issue on Github at: github.com/inirt/inirt/issues
or email me at mjkleinsa@gmail.com

Functions: Overview

This section explains the way the inirt package's functions work. For tutorials, see the previous sections.

Overview of the inirt package's functions

The most important function from inirt is the inirt function: 

inirt::inirt(model, data, ...)
Where model is an inirt model, data is inirt, data, ... etc

NOTE: if you library the package first, then one only needs the call the function (no :: required): 

library(inirt)
inirt(model, data, ...)

inirt::inirt(model, data, ...)
